# Dining Images

This folder contains placeholder images used by the dining page (`src/pages/Dining.jsx`).

## Replace with real photos

To use the provided pictures, replace the placeholder SVGs with your actual image files and keep the same filenames:

- `dining-main.svg` → (main restaurant photo)
- `dining-terrace.svg` → (sunrise terrace photo)
- `dining-bar.svg` → (bar & light dining photo)

If you prefer to use JPEG/PNG, you can also update `src/pages/Dining.jsx` to import the corresponding file extensions.
